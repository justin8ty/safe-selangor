# Safe Selangor
This is a repo for the Safe Selangor project.

### Getting Started
Before you begin, ensure Node.js is installed. (version 18.17.0 or later)
```bash
node -v
```
- If not installed, install it from [nodejs.org](https://nodejs.org/)

Clone the repo & cd into the repo:
```bash
git clone <repo-url>
cd safe-selangor
```

Start the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
